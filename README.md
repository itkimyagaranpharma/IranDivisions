## فارسی

این فایل شامل تقسیمات کشوری جمهوری اسلامی ایران (سال ۱۴۰۳) است که از وب‌سایت رسمی مرکز آمار ایران دریافت شده است.

منبع:
https://amar.org.ir/statistical-information/statid/55522

این فایل شامل دو شیت است:

### TAGHSIMAT_1403
نسخه اصلی داده‌های دریافت‌شده از مرکز آمار ایران بدون هیچ‌گونه تغییر.

### TAGHSIMAT_1403_edited
نسخه ویرایش‌شده که تنها اصلاحات زیر در آن اعمال شده است:

- فرادبنه → فرادنبه (اصلاح خطای تایپی)
- چاه بهار → چابهار (نام رسمی مورد استفاده از سال ۱۳۸۰)
- هفتگل → هفتکل (اصلاح نام رسمی شهرستان)

## English

This file contains the Administrative Divisions of Iran (1403/2024), obtained from the official website of the Statistical Center of Iran.

Source:
https://amar.org.ir/statistical-information/statid/55522

The workbook contains two worksheets:

### TAGHSIMAT_1403
Original dataset downloaded from the Statistical Center of Iran without any modifications.

### TAGHSIMAT_1403_edited
Revised version containing only the following corrections:

- فرادبنه → فرادنبه (Typographical correction)
- چاه بهار → چابهار (Official place name adopted since 2001)
- هفتگل → هفتکل (Official county name correction)
